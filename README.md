## ✨ Features

- 🖥️ **Command Center** — Live CPU, GPU, RAM & Network telemetry in real-time
- ⚡ **One-Click Boost** — Instantly optimize system performance
- 🎮 **Gaming Mode** — Switch to Ultimate power plan for max FPS
- 🌐 **Network Optimizer** — DNS switcher, TCP booster, ping monitor & IP info
- 🧹 **System Cleaner** — Remove junk files, clear cache & free up disk space
- 🔒 **License System** — 7-day free trial + paid plans via Supabase
- 🔄 **Auto-Updater** — Silent background updates via GitHub Releases
- 🗂️ **System Tray** — Runs in background with tray icon & tooltip stats
